i2c_engine_usci.c File Reference

Detailed Description

I2C (USCI Implementation) for HIDI2C Specification I2C engine for the HIDI2C implementation.

This is the transport layer of the API. In this implementation, USCI is used for management of the I2C. This layer is abstracted out so another implementation of I2C can be used.

GPIO ports and specific interface settings are defined in the hidi2c_settings.h file.

#include <msp430.h>
#include <hidi2c_settings.h>
#include <hidi2c_errno.h>
#include "hid_engine.h"
#include <i2c_engine_usci.h>
#include <i2c_engine_usci_init.h>
#include <stdlib.h>
#include <string.h>
#include <stdio.h>
