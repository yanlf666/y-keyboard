i2c_engine_usci_init.c File Reference

Detailed Description

Contains all of the initialization functions and structures for the I2C layer of the HIDI2C stack. ifdefs are used to minimize the memory footprint for users not using multiple devices. Specific USCI settings are defined in the hidi2c_settings.h file.

#include <hidi2c_types.h>
#include <i2c_engine_usci_init.h>
#include <hidi2c_settings.h>
#include <msp430.h>
